# Dummy Shop Project

## Run Backend

cd server
npm install
node server.js

Server: http://localhost:3000

---

## Run Frontend

npm install
npm run dev

Frontend: http://localhost:5173

---

## Database

* Use db/init.sql to recreate database
* Or use db/app.db directly

---

## Features

* Product & Category CRUD
* Image upload + resize
* Dynamic product display
* Shopping cart (AJAX + localStorage)
* Cart persists after refresh
